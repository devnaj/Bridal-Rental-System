# Bridal Rental System

A modular full-stack web application built using Django to streamline and manage bridal boutique inventories, customer bookings, and attire rental tracking.

## 🚀 Features
- **Inventory Management:** Track unique bridal attire, sizing, condition, and availability status.
- **Rental Tracking:** Monitor booking dates, return deadlines, and rental history.
- **Automated Seeding:** Built-in scripting to quickly populate the database with demo inventory and setup configurations.

## 🛠️ Tech Stack
- **Backend:** Python, Django
- **Database:** SQLite (default development database)
- **Frontend:** Django Templates, HTML, CSS

## 📁 Project Structure
Based on the main repository structure:
- `bridal_inventory/` — Application handling attire catalogs, categorization, and item stock metrics.
- `rentals/` — Application managing customer reservations, rental terms, and scheduling details.
- `bridalsystem/` — Core project configuration, global settings, and routing files.
- `seed.py` — Database population script containing dummy datasets for rapid local deployment.
- `manage.py` — Django management command-line utility.
